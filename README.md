# HEIC

JPEG 용량은 너무 크고 내 iPhone, iPad, iCloud 용량은 너무 작은데, 더 큰 용량을 구입하기에는 내 주머니가 너무 가볍다.  

## 사용방법

### Terminal

```shell
heic (options) [images...]  
```

- Options
    * -q 압축 품질 (0 ~ 100, 기본: 65)
